- `API_HASH` API_HASH Anda dari my.telegram.org
- `API_ID` API_ID Anda dari my.telegram.org
- `BOT_TOKEN` Token Bot Anda dari @BotFather
- `REMOVEBG_API` REMOVEBG_API Anda dari removebg.com/api

</details>
